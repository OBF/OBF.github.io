# GSoC 2026: Call for Projects, Mentors, and Co-Admins

The Open Bioinformatics Foundation (OBF) plans to participate in Google Summer of Code (GSoC) 2026 and invites community members to propose project ideas, volunteer as mentors, or help as co-administrators. Per the official [GSoC 2026](https://summerofcode.withgoogle.com/programs/2026) timeline, organization applications close on February 3. ## About Google Summer of Code

Google Summer of Code is a global program that brings students and new contributors together with open-source organizations to work on well-scoped projects under mentor guidance. The focus is on meaningful open-source contributions, strong mentorship, and long-term community impact.

## Expectations

### Projects

* Clearly scoped and feasible within the GSoC timeline
* Address real problems or extend existing open-source systems
* Define clear deliverables and milestones

### Mentors

* Provide technical guidance and regular feedback
* Review code and help contributors succeed
* Commit consistent time during the program

### Co-Admins

* Support org application and coordination
* Help manage mentors, contributors, and timelines
